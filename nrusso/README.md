# nrusso
Sylvain Schmitt
Dec 1, 2025

Analyses developed during the the post doc of Nicholas Russo at Harvard
on modelling the effect of defaunation in Central Africa on forest
dynamics with TROLL.

**nrusso** includes:

- `20250422.qmd` examples for TROLL model evaluation
- `cd.qmd` analyses on crown depth modelling
- `cr.qmd` analyses on crown radius modelling
