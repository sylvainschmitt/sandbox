# vbadouard
Sylvain Schmitt
Dec 1, 2025

Analyses developed during the doctorate of Vincyane Badouard at AMAP on
canopy and undrestory species light niche in French Guiana.

**vbadouard** includes:

- `residuals.qmd` analyses on Binomial pseudo-residuals
- `vincyane.qmd` analyses on background data
- `vincyane2.qmd` analyses on parameters 3-D space
- `vincyane3.qmd` analyses on rational behind the model
